ecommended Prechecks for FastAPI Projects
Essential checks:

Black - Automatic code formatting
isort - Import statement organization
Ruff - Fast Python linter (catches bugs, style issues)
MyPy - Static type checking (important for FastAPI's type hints)
Bandit - Security vulnerability scanning
Trailing whitespace & EOF fixes - Clean code hygiene

Additional valuable checks:

pytest - Run your test suite
Coverage - Ensure test coverage meets threshold
Poetry/pip check - Validate dependencies
Safety - Check for known security vulnerabilities in dependencies
